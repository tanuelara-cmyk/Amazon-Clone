# 🛒 Amazon Homepage Clone

A responsive Amazon homepage clone built using **HTML5** and **CSS3**.

## 🚀 Live Demo

🔗 https://tanuelara-cmyk.github.io/Amazon-Clone/

## 📌 Features

- Responsive navigation bar
- Hero banner section
- Product category cards
- Hover effects
- Clean and modern UI

## 🛠️ Technologies Used

- HTML5
- CSS3
